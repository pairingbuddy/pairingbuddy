# identify-coverage-gaps

**Skill:** @mimer-code:identify-coverage-gaps

**Purpose:** Analyze existing tests and code to identify missing test scenarios.

**When to spawn:**
- In COVERAGE phase after initial tests are implemented
- Pass: existing tests, production code, requirements

**What it returns:** Structured list of missing test scenario descriptions organized by category - empty if no gaps

**What it does NOT do:** Write tests, implement code, call other skills
