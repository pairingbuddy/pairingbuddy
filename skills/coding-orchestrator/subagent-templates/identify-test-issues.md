# identify-test-issues

**Skill:** @pairingbuddy:identify-test-issues

**Purpose:** Analyze test code and return structured report of quality issues.

**When to spawn:**
- In REFACTOR phase before refactoring tests
- Pass: test code to analyze

**What it returns:** Structured report of issues organized by category (Test Quality, Test Structure, Test Pyramid, Test Performance, Test Patterns) - empty if no issues

**What it does NOT do:** Fix issues, refactor code, call other skills
