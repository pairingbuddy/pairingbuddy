# identify-code-issues

**Skill:** @pairingbuddy:identify-code-issues

**Purpose:** Analyze production code and return structured report of quality issues.

**When to spawn:**
- In REFACTOR phase before refactoring production code
- Pass: production code to analyze

**What it returns:** Structured report of issues organized by category (Code Quality, Design, Performance, Security, Code Patterns) - empty if no issues

**What it does NOT do:** Fix issues, refactor code, call other skills
