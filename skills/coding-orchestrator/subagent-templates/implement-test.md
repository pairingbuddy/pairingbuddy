# implement-test

**Skill:** @pairingbuddy:implement-test

**Purpose:** Write ONE failing test for a scenario and verify it fails correctly.

**When to spawn:**
- In RED phase when implementing a test scenario
- Pass: test scenario description, how to run tests

**What it returns:** Test written and verified failing for the right reason (not syntax error)

**What it does NOT do:** Write production code, refactor tests, call other skills
