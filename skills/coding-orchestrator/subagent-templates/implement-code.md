# implement-code

**Skill:** @pairingbuddy:implement-code

**Purpose:** Write minimal production code to make ONE failing test pass and verify all tests pass.

**When to spawn:**
- In GREEN phase after a test is failing
- Pass: failing test context, how to run tests

**What it returns:** Minimal code written, test passes, all tests pass

**What it does NOT do:** Refactor code, optimize code, add untested features, call other skills
